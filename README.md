# workmate-access

Workmate Acces ist ein Tool für die Verwaltung und Controlling unser zugangssystem

## Quickstart

- (optional) `.env` aus `.env.example` kopieren
- venv aktivieren: `source .venv/bin/activate`
- starten: `make run` (oder `uvicorn app.main:app --reload --port 8000`)

## Navigation
-   → direkt ins Projekt springen
-         → FZF Projekt-Picker
-    → via zoxide, nachdem du das Verzeichnis genutzt hast
